# Everyday Habit Tracker

Everyday Habit Tracker is a simple and interactive web app to track daily habits and maintain consistency. User can add habits, mark them as done or skipped, and monitor daily progress using a calendar-based UI.

---

## Features

- Add, edit, and delete habits
- Mark habits as done or skip
- Calendar-based habit tracking
- Daily completed habit count
- Data persistence using localStorage

---

## Tech Stack

- TypeScript
- HTML5
- CSS3
- LocalStorage API
